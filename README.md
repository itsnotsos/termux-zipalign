# termux-zipalign

## Prebuilt zipalign for termux android

#### List of supported architectures:

- [x] arm64-v8a
- [x] armeabi-v7a
- [x] i686
- [x] x86_64

#### Install:
``` console
$ curl -s https://raw.githubusercontent.com/rendiix/rendiix.github.io/master/install-repo.sh | bash
$ pkg install zipalign
```

#### Remove:
``` console
$ apt remove zipalign
```
